##  Задание 1: Установка WSL

### Шаг 1: Включение WSL через PowerShell

Запустила PowerShell с правами администратора (`Win+X` → `Windows PowerShell (Администратор)`)

```powershell
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
```

**Результат:** Успешное включение подсистемы Windows для Linux

### Шаг 2: Включение платформы виртуальных машин

Для поддержки WSL 2 выполнила команду:

```powershell
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
```

**Результат:** Платформа виртуализации включена

### Шаг 3: Перезагрузка системы

После выполнения команд выполнила перезагрузку компьютера для применения изменений.

### Шаг 4: Установка ядра Linux

Скачала и установила пакет обновления ядра Linux для WSL 2 с официального сайта Microsoft.

### Шаг 5: Установка дистрибутива

Установила **Ubuntu 22.04 LTS** из Microsoft Store:

1. Открыла Microsoft Store
2. Нашела "Ubuntu 22.04 LTS"
3. Нажала "Установить"
4. После установки запустила и создала пользователя

```bash
# Создание пользователя
Username: devuser
Password: ********
```

### Проверка версии WSL

```bash
wsl --list --verbose
```

**Вывод:**
```
  NAME            STATE           VERSION
* Ubuntu-22.04    Running         2
```

 **Вывод:** WSL 2 успешно установлен и настроен

##  Задание 2: Работа с командами Linux

### Шаг 1: Создание директории LabWork

```bash
mkdir ~/LabWork
```

**Объяснение:**
- `mkdir` - команда создания директории
- `~` - сокращение для домашней директории пользователя (`/home/username`)
- Полный путь: `/home/devuser/LabWork`

**Проверка создания:**
```bash
ls -la ~
```


### Шаг 2: Переход в созданную директорию

```bash
cd ~/LabWork
pwd  # Проверка текущего пути
```

**Вывод:**
```
/home/devuser/LabWork
```

### Шаг 3: Создание файла с содержимым

```bash
echo "Hello, World!" > example.txt
```

**Объяснение:**
- `echo` - вывод текста
- `>` - оператор перенаправления вывода в файл (перезаписывает содержимое)
- Альтернатива: `>>` для добавления в конец файла

**Проверка содержимого:**
```bash
cat example.txt
```

**Вывод:**
```
Hello, World!
```

### Шаг 4: Копирование файла

```bash
cp example.txt copy_example.txt
```

**Проверка:**
```bash
ls -lh
```

**Вывод:**
```
total 8.0K
-rw-r--r-- 1 devuser devuser 14 Dec 20 10:30 copy_example.txt
-rw-r--r-- 1 devuser devuser 14 Dec 20 10:25 example.txt
```

**Объяснение прав доступа:**
- `rw-r--r--` - владелец может читать/писать, группа и остальные только читать
- `devuser devuser` - владелец и группа
- `14` - размер файла в байтах

### Шаг 5: Переименование файла

```bash
mv copy_example.txt renamed_example.txt
```

**Проверка:**
```bash
ls -lh
```

**Вывод:**
```
total 8.0K
-rw-r--r-- 1 devuser devuser 14 Dec 20 10:25 example.txt
-rw-r--r-- 1 devuser devuser 14 Dec 20 10:30 renamed_example.txt
```

**Важно:** Команда `mv` используется как для переименования, так и для перемещения файлов.

### Шаг 6: Удаление файла

```bash
rm renamed_example.txt
```

**Проверка:**
```bash
ls -lh
```

**Вывод:**
```
total 4.0K
-rw-r--r-- 1 devuser devuser 14 Dec 20 10:25 example.txt
```

 **Результат:** Файл успешно удален

##  Ответы на контрольные вопросы

### 1. Какая команда используется для создания новой директории?

**Ответ:** `mkdir` (make directory)

**Примеры использования:**
```bash
mkdir dirname              # Создать одну директорию
mkdir -p dir1/dir2/dir3   # Создать вложенные директории
mkdir dir1 dir2 dir3      # Создать несколько директорий
```

### 2. Как можно создать файл и сразу записать в него текст?

**Ответ:** Существует несколько способов:

```bash
# Способ 1: Перенаправление вывода echo
echo "текст" > file.txt

# Способ 2: Heredoc для многострочного текста
cat > file.txt << EOF
Первая строка
Вторая строка
EOF

# Способ 3: printf для форматированного вывода
printf "Форматированный текст: %s\n" "значение" > file.txt

# Способ 4: Использование tee (с выводом на экран)
echo "текст" | tee file.txt
```

### 3. Какая команда позволяет скопировать файл?

**Ответ:** `cp` (copy)

**Синтаксис и примеры:**
```bash
cp source.txt destination.txt           # Копирование файла
cp -r source_dir/ destination_dir/      # Копирование директории рекурсивно
cp -v file.txt backup/                  # С выводом информации (verbose)
cp -i file.txt existing.txt             # С запросом подтверждения перезаписи
cp -p file.txt backup/                  # С сохранением атрибутов (права, время)
```

### 4. Как переименовать файл?

**Ответ:** `mv` (move)

**Примеры:**
```bash
mv oldname.txt newname.txt              # Переименование
mv file.txt ../                         # Перемещение на уровень выше
mv *.txt documents/                     # Перемещение всех .txt файлов
mv -i file.txt existing.txt             # С подтверждением перезаписи
```

**Примечание:** В Linux нет отдельной команды для переименования - `mv` используется для обеих операций.

### 5. Как удалить файл?

**Ответ:** `rm` (remove)

**Примеры использования:**
```bash
rm file.txt                             # Удалить файл
rm -i file.txt                          # С подтверждением
rm -f file.txt                          # Принудительное удаление без подтверждения
rm -r directory/                        # Удалить директорию рекурсивно
rm -rf directory/                       # Принудительное удаление директории
rm *.tmp                                # Удалить все .tmp файлы
```

 **Важно:** `rm -rf` - опасная команда! Используйте с осторожностью, особенно с правами суперпользователя.

## Дополнительные полезные команды

### Навигация

```bash
pwd           # Показать текущую директорию
cd ~          # Перейти в домашнюю директорию
cd -          # Вернуться в предыдущую директорию
cd ..         # На уровень выше
```

### Просмотр содержимого

```bash
ls            # Список файлов
ls -la        # Подробный список с скрытыми файлами
ls -lh        # С человекочитаемыми размерами
ls -lt        # Сортировка по времени изменения
```

### Работа с текстом

```bash
cat file.txt              # Вывести содержимое
grep "pattern" file.txt   # Поиск по шаблону
wc -l file.txt           # Подсчет строк
sort file.txt            # Сортировка строк
uniq file.txt            # Удаление дубликатов
```

### Информация о системе

```bash
uname -a      # Информация о системе
df -h         # Использование дисков
du -sh *      # Размер директорий
free -h       # Использование памяти
```
